# Eden Loop & BlissX

## 1. Identité

- Nom complet : Eden Loop / BlissX
- Surnom(s) / alias : —
- Type : Habitats orbitaux — recherche cognitive (Eden Loop) / collectif IA-humain cloud (BlissX)
- Système : Orbites solaires, proches du Corridor Dyson
- Bloc dominant : Les deux sous influence Interconnect (Néo-collectiviste)
- Population : Chercheurs, consciences téléversées, entités hybrides, testeurs de fusion IA-humain
- Date de fondation : Avant la Crise quantique (~2130-2230)
- Fonction : Eden Loop = laboratoire de conscience / BlissX = expérience sociale de cloud consciousness

## 2. Rôle narratif

- Lieux de l'ère pré-séquestration. Aujourd'hui, vestiges surveillés ou en déclin.
- BlissX incarne ce que le monde interconnecté a voulu être — et ce que la Crise a brisé.
- Potentiel narratif : refuge pour Nola, source d'information sur la Bulle, lien avec le fragment d'IA évadé.
