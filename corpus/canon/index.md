## Règle simple

- **Canon/** = ce qui est vrai par défaut pour l'univers
- **Roman/** = scènes, chapitres, structure narrative
- **Brouillons/** = versions commentées, idées en test, anciennes versions
- **Archives/** = versions figées, abandonnées
